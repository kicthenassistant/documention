◆ sendMMI()

function async sendMMI ( final_transcript)

Processa o final_transcript da fala, converte-o(SPEECH TO TEXT - GOOGLE CLOUD) e envia para o cliente MMI.

  • 1º: Converte números falados nas suas representações digitais.
  • 2º: Cria um objeto para enviar ao servidor, que inclui o final_transcript processado pelo RASA.
  • 3º: Constrói diferentes mensagens, dependendo da inteção reconhecida pelo RASA e envia-as para o cliente MMI através do método sendToIM.
Parameters
{string}final_transcript - O final_transcript recebido do reconhecimento de voz (SPEECH TO TEXT - GOOGLE CLOUD).
Returns
{Promise} Uma promessa que é resolvida quando a interação MMI é tratada com sucesso.
